Odoo Website Social

Extend Social Networks on Odoo.

What it is?

If you need add more Social Network to your Website, this is a good option. This module allow you load more Social Networks to Website Odoo throught Website Administrator.

How use it?

Just load your Social Networks on Website Administrator > Settings and the module will put it on your website.

Social Networks Supported

Bitbucket, Instagram, Vimeo, Twitch, Tumblr, Flickr, Reddit, WhatsApp (with WhatsApp API) and Skype (with Skype API).
